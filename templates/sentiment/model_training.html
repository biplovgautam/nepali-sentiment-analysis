Parameter Guide
Alpha (Smoothing)
  • 1.0 (Default): Standard Laplace smoothing
  • 0.1-0.5: Less smoothing, more sensitive to training data
  • 2.0-5.0: More smoothing, better generalization
  • Effect: Prevents zero probabilities
Test Size
  • 0.2 (Default): 80% train, 20% test
  • 0.1: More training data, less test data
  • 0.3: Less training data, more test data
  • Effect: Balances training vs evaluation
Cross-Validation Folds
  • 5 (Default): Good balance of reliability and speed
  • 3: Faster, less reliable estimate
  • 10: More reliable, slower computation
  • Effect: Validates model robustness
Recommendation: Start with default values for your first training. You can experiment with different parameters later to optimize performance.
