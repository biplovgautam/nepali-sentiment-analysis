Filtering Tips
Balanced Dataset Benefits:
  • Prevents model bias towards dominant classes
  • Improves overall model performance
  • More reliable evaluation metrics
  • Better generalization to new data
Recommendations:
  • Use equal samples per class when possible
  • Minimum 50-100 samples per class for training
  • Consider your computational resources
  • More data generally leads to better models
Note: After filtering, the system will randomly sample the specified number of instances from each class to create a balanced dataset for training.
